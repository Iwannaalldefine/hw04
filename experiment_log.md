# ASR 实验过程与结果记录

## 1. 实验环境说明
* **操作系统：** Windows
* **Python 版本：** Python 3.12.0
* **硬件加速：** 无 GPU 依赖，纯 CPU 离线运行
* **ASR 框架：** Vosk (`pip install vosk`)
* **所用模型：** `vosk-model-small-cn-0.22` (约 42MB)

## 2. 测试音频说明
使用剪映克隆个人声音生成的配音音频，时长约 1 分钟。经过 Python `pydub` 库预处理，格式已统一转换为 16000Hz 采样率、单声道、16-bit 宽度的标准 WAV 格式 (`audio_sample.wav`)。

## 3. 识别样例对比

**【Ground Truth 原文本】**
人脸识别技术作为人工智能领域最成熟的应用之一，其核心在于通过深度卷积神经网络(CNN)提取面部特征。系统首先在图像中定位人脸区域，随后通过对关键特征点——如眼睛间距、鼻梁高度和唇部轮廓的精确捕捉，将其转化为一组高维向量。通过将该向量与数据库中的已知数据进行欧氏距离或余弦相似度比对，即可实现秒级的身份验证。如今，人脸识别已从简单的考勤门禁，演进到金融支付级别的活体检测和公共安全领域的大规模检索。虽然环境光照、遮挡以及人脸角度的变化曾是巨大的技术挑战，但随着残差网络(ResNet)等架构的普及，识别准确率已得到显著提升。未来，如何在提升算法精度的同时，平衡数据隐私与伦理边界，将是该技术发展的核心议题。

## 4. 延迟与错误率观察
1. **推理速度：** 极快。得益于轻量级模型，CPU 加载模型后几乎是秒级吐出识别文字，没有明显的计算延迟。
2. **错误率分析：** 整体连贯性较好，基础中文词汇识别准确。但由于使用的是 40MB 的极小模型，缺乏大语言模型的上下文纠错能力，在处理同音字和**中英夹杂的专业词汇**时出现了典型的识别错误。例如：
   * 英文缩写 `CNN` 被音译识别为 **“西安”**。
   * 英文 `ResNet` 被音译识别为 **“微信那”**。
   * “如眼睛间距”连读被识别为 **“揉眼睛兼具”**。
   这些错误符合轻量级纯声学模型的预期表现。
