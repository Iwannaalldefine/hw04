# 剪映声音克隆与导出说明

## 克隆步骤概要
1. 打开剪映专业版（PC端），点击开始创作，进入创作界面，新建文本轨道，输入大模型生成的“人脸识别技术”文案。
2. 选中该文本，在右上角面板选择“朗读”标签页。
3. 点击“克隆音色”，按照系统提示录制个人声音样本进行云端训练。
4. 训练完成后，应用该专属音色为文本生成配音。（受限于硬件设备坏了，尝试了克隆，效果不理想，最终使用了现存的配音音色）

## 导出与格式处理
* 初始导出格式：导出纯音频文件（格式为 `.wav`，参数为 44.1kHz, 立体声）。
* 格式标准化：由于 Vosk 引擎严格要求音频格式，在 `asr_code` 目录下编写了 Python 脚本（利用 `pydub` 库），将导出的音频强制转换为 **16kHz 采样率、单声道 (Mono)、16-bit** 的标准 WAV 格式。转换后的文件命名为 `audio_sample.wav`，供后续 ASR 程序直接调用。

*(注：由于音频文件体积较大，已直接存放在本地 `asr_code` 目录中用于代码复现。)*
